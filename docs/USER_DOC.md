## 🚀 快速开始

### 1. 访问面板

- **本地使用**: 直接打开 `index.html`
- **在线版本**: https://zhanggefei.github.io/opendashboard/

### 2. 添加任务

点击任意任务列表顶部的 **"+ 添加任务"** 按钮，填写：
- 任务标题（必填）
- 任务描述
- 优先级（P0/P1/P2）
- 负责人
- 截止日期

### 3. 管理任务状态

拖动任务卡片在不同状态区域间移动：
- 🆕 待办 → 🔄 进行中 → ✅ 已完成
- ⏸️ 阻塞：遇到阻碍时

## 🎯 核心功能

### 任务管理

| 功能 | 操作 |
|------|------|
| 创建任务 | 点击"+ 添加任务" |
| 编辑任务 | 点击任务卡片 |
| 删除任务 | 右键菜单 → 删除 |
| 调整顺序 | 拖动任务卡片 |
| 批量操作 | 勾选多选 → 批量操作栏 |

### 筛选与搜索

- **关键词搜索**: 支持标题、描述模糊搜索
- **状态筛选**: 按任务状态过滤
- **优先级筛选**: P0/P1/P2
- **负责人筛选**: 按负责人过滤
- **组合筛选**: 多条件同时生效

### 数据统计

顶部统计卡片显示：
- ✅ 已完成任务数
- 🔄 进行中任务数
- 🆕 待办任务数
- ⏸️ 阻塞任务数

点击统计卡片可快速筛选对应状态的任务。

## ⚡ 高级功能

### 1. 任务模板

快速创建常用任务类型：
1. 点击 📋 模板按钮
2. 选择预定义模板
3. 自动填充任务信息

**内置模板**:
- 客户跟进
- 产品上线
- Bug 修复
- 会议准备

### 2. 时间追踪

记录任务实际耗时：
- 点击任务卡片 → 启动计时器
- 自动记录开始/结束时间
- 累计工时统计

### 3. 任务依赖

设置任务前后置关系：
- 编辑任务 → 依赖关系
- 选择前置任务
- 自动阻塞未完成依赖的任务

### 4. 子任务

拆解复杂任务：
- 编辑任务 → 子任务
- 添加子任务列表
- 独立跟踪每个子任务状态

### 5. 评论与日志

团队协作：
- @提及 团队成员
- 评论历史记录
- 自动活动日志

### 6. 标签系统

多维度分类：
- 创建自定义标签
- 多标签关联
- 按标签筛选

### 7. 批量操作

高效处理多任务：
1. 勾选多个任务
2. 批量状态变更
3. 批量删除

### 8. 导入导出

数据备份与迁移：
- **JSON**: 完整数据导出
- **CSV**: Excel 兼容格式
- **PDF**: 美观报表

### 9. 通知系统

及时提醒：
- 截止日期提醒
- 任务分配通知
- 免打扰模式

### 10. 深色模式

护眼模式：
- 点击 🌙 按钮切换
- 自动记住偏好

### 11. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `N` | 新建任务 |
| `Ctrl/Cmd + F` | 搜索 |
| `Ctrl/Cmd + S` | 保存 |
| `Delete` | 删除选中任务 |
| `Esc` | 关闭弹窗 |

### 12. 移动端手势

触屏优化：
- **左滑**: 删除任务
- **右滑**: 完成任务
- **长按**: 编辑任务

### 13. 语音输入

语音创建任务：
- 点击 🎤 按钮
- 说出任务内容
- 自动识别并创建

### 14. 性能监控

实时性能指标：
- 点击 ⚡ 按钮查看
- 页面加载时间
- 渲染耗时
- API 响应时间
- 内存使用

### 15. 身份管理

多身份切换：
- 点击 👤 按钮
- 管理多个身份
- 按身份筛选任务

## ❓ 常见问题

### Q: 数据保存在哪里？

A: 任务数据保存在 `tasks/tasks.json` 文件中。可以手动编辑或版本控制。

### Q: 如何备份数据？

A: 使用导出功能（📥 按钮）导出 JSON 文件，或复制 `tasks/` 文件夹。

### Q: 支持多人协作吗？

A: 支持！通过 Git 共享任务文件，或使用 OpenClaw API 集成。

### Q: 如何重置所有数据？

A: 删除 `tasks/tasks.json` 文件，刷新页面会重新加载示例数据。

### Q: 性能监控显示的数据准确吗？

A: 性能数据来自浏览器 Performance API，实时反映当前页面性能。

### Q: 语音输入支持哪些语言？

A: 依赖浏览器语音识别，通常支持系统设置的语言。
