## 🚀 快速入门

### 1. 访问面板

- **在线版本**: https://zhanggefei.github.io/opendashboard/
- **本地运行**: 直接打开 `index.html` 文件

### 2. 基本操作

1. **查看任务**: 任务按状态分组显示（进行中、待办、阻塞、已完成）
2. **创建任务**: 点击右上角 `➕ 新建任务` 按钮
3. **编辑任务**: 点击任务卡片进入编辑模式
4. **完成任务**: 点击任务卡片上的 ✅ 按钮

### 3. 筛选搜索

- **搜索框**: 输入关键词搜索任务标题和描述
- **状态筛选**: 选择特定状态的任务
- **优先级筛选**: 按 P0/P1/P2 筛选
- **负责人筛选**: 查看特定人员的任务

## 🎯 核心功能

### 任务管理

| 功能 | 说明 |
|------|------|
| 创建任务 | 填写标题、描述、优先级、负责人等信息 |
| 编辑任务 | 修改任务的任何字段 |
| 删除任务 | 删除任务（可回收站恢复） |
| 任务状态 | 待办 → 进行中 → 已完成 / 阻塞 |
| 优先级 | P0(紧急重要) / P1(重要) / P2(普通) |

### 批量操作

1. 勾选任务前的复选框
2. 底部出现批量操作栏
3. 可批量：
   - 更改状态
   - 更改负责人
   - 删除任务

### 任务模板

**使用模板快速创建任务**:

1. 点击 `📋` 按钮打开模板面板
2. 选择预设模板或创建新模板
3. 点击模板自动填充任务信息

**预设模板**:
- 📝 日常任务
- 🐛 Bug 修复
- ✨ 新功能开发
- 📊 数据分析
- 📞 客户跟进

## ⚡ 高级功能

### 1. 身份管理

**切换不同身份查看任务**:

1. 点击 `👤` 按钮打开身份面板
2. 选择不同身份（如：销售、技术、管理等）
3. 任务列表自动筛选显示对应身份的任务

### 2. 任务依赖

**设置任务前后置依赖**:

1. 编辑任务时选择"依赖关系"
2. 选择前置任务
3. 前置任务完成后，当前任务自动解锁

### 3. 时间追踪

**记录任务耗时**:

1. 点击任务卡片上的 ⏱️ 按钮开始计时
2. 再次点击停止计时
3. 自动累计任务总耗时

### 4. 标签系统

**给任务添加标签**:

1. 编辑任务时添加标签
2. 支持多标签
3. 可按标签筛选任务

**常用标签**:
- `#紧急` `#重要` `#等待` `#会议` `#客户`

### 5. 活动日志

**查看任务变更历史**:

1. 每个任务都有活动日志
2. 记录所有状态变更、评论、附件等操作
3. 可追溯任务完整生命周期

### 6. 通知系统

**设置任务提醒**:

1. 编辑任务时设置截止日期
2. 开启提醒通知
3. 支持多渠道通知（飞书、邮件等）

### 7. 数据导出

**导出任务数据**:

1. 点击 `📥` 按钮
2. 选择导出格式：
   - **JSON**: 完整数据备份
   - **CSV**: Excel 表格格式
   - **PDF**: 美观报表格式
3. 可选择筛选条件导出

### 8. 数据报表

**查看统计分析**:

1. 点击 `📊` 按钮打开报表
2. 查看：
   - 任务状态分布图
   - 优先级分布图
   - 负责人任务量
   - 完成进度趋势

### 9. 移动端手势

**手机端快捷操作**:

- **左滑**: 删除任务
- **右滑**: 完成任务
- **长按**: 编辑任务

### 10. 语音输入

**语音快速创建任务**:

1. 点击 `🎤` 按钮
2. 说出任务内容
3. 自动识别并创建任务

### 11. AI 智能建议

**获取任务优先级建议**:

1. 系统分析任务紧急程度
2. 自动推荐处理顺序
3. 智能识别重要任务

### 12. 性能监控

**查看系统性能**:

- 页面加载时间
- 任务渲染耗时
- API 调用耗时
- 内存使用情况

## ❓ 常见问题

### Q1: 任务数据保存在哪里？

任务数据保存在浏览器本地存储（LocalStorage），也可导出备份。

### Q2: 如何备份数据？

使用 `📥 导出数据` 功能，选择 JSON 格式定期备份。

### Q3: 可以多人协作吗？

支持团队协作功能，可分配任务给不同成员。

### Q4: 如何恢复删除的任务？

删除的任务进入回收站，可在回收站中恢复。

### Q5: 支持自定义字段吗？

目前支持标准字段，自定义字段功能开发中。
