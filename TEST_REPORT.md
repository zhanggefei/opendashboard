## 🐛 Bug 记录

### Bug #1: 身份切换任务数量不更新 ✅ 已修复

**问题描述**: 点击身份卡片切换时，下面的任务卡片数量没有变化

**影响范围**: 身份管理模块

**严重程度**: 🔴 高

**复现步骤**:
1. 打开身份管理面板
2. 点击不同的身份卡片
3. 观察任务数量显示

**预期结果**: 任务数量应该随身份切换而变化

**实际结果**: 任务数量保持不变

**修复方案**:
1. 统一身份 ID 为纯英文（dogdan 代替 dog 蛋）
2. 修复 load 函数，正确合并保存的任务数据
3. 确保 addSampleTasks 使用正确的 ID

**修复状态**: ✅ 已修复，已部署

**修复时间**: 2026-03-01 20:35
