## 🐛 Bug 记录与修复

### Bug #001 - 初始化顺序错误 ✅ 已修复
**发现时间**: 2026-03-02 15:35  
**严重程度**: 🔴 严重 (导致页面无法加载)  
**影响范围**: 所有功能模块  

**问题描述**:  
在 app.js 文件开头，在所有类定义之前就尝试初始化实例：
```javascript
// ❌ 错误写法
window.dependencyManager = new TaskDependencyManager(); // 类还未定义！
```

**修复方案**:  
将所有初始化移到 DOMContentLoaded 事件监听器中：
```javascript
// ✅ 正确写法
document.addEventListener('DOMContentLoaded', () => {
    window.dependencyManager = new TaskDependencyManager();
});
```

**提交**: b29da6f  
**验证**: ⏳ 待验证 (等待 GitHub Pages 部署)
